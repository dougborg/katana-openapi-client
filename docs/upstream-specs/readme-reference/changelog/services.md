# Services

Services are a type of item, similar to Products and Materials, except a service is a non-physical good that is always available that doesn't participate in warehouse workflows (picking, packing, receiving, put-away, moving).

With the release of [Services](https://developer.katanamrp.com/reference/the-service-object), we’re also rolling out endpoints for managing them via the API.\
The following endpoints are now available:

* [GET api.katanamrp.com/v1/services](https://developer.katanamrp.com/reference/getallservices)
* [GET api.katanamrp.com/v1/services/\{id}](https://developer.katanamrp.com/reference/getservice)
* [POST api.katanamrp.com/v1/services](https://developer.katanamrp.com/reference/createservice)
* [PATCH api.katanamrp.com/v1/services/\{id}](https://developer.katanamrp.com/reference/updateservice)
* [DELETE api.katanamrp.com/v1/services/\{id}](https://developer.katanamrp.com/reference/deleteservice)
