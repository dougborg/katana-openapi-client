# Introduction

Welcome to the Katana API documentation.

Use the **Resources panel** to explore detailed information and code samples for every public endpoint. In the top-right corner of each endpoint section, you can switch between example requests in **cURL**, **Node.js**, **Go**, **Ruby**, and **Python**.

The Katana API is fully described using the **OpenAPI 3.0** specification, which you can access <Anchor label="here" target="_blank" href="https://api.katanamrp.com/v1/openapi.json">here</Anchor> to review the complete schema and endpoint definitions.

To learn more about using Katana as a product, visit our **<Anchor label="Knowledge Base" target="_blank" href="https://support.katanamrp.com/">Knowledge Base</Anchor>**, where you’ll find step-by-step guides, tutorials, and video walkthroughs.

You can also stay up to date with the latest product updates and announcements <Anchor label="here" target="_blank" href="https://katanamrp.com/news/">here</Anchor>.
